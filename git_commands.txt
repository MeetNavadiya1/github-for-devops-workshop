git init

git add . -> to stage all the files

git add testing.py -> to stage sing files

git rm --cache testing.py -> to make untracked file from staged.

git commit-m "added file" -> to track the file

git restore testing.py -> to recover or get deleted files

git remote -v -> to see remote repository

git branch -M main -> to set branch name mater to main in local

git push -u origin main -> to upstream new branch and push code to the github in main branch.

git branch -> to list branches

git branch dev -> to create new branch

git checkout -b dev -> to create branch and switch into it.

git log / git log --oneline -> to see the logs of git

git pull origin main -> to pull from a specific branch

git fetch -> to get remote branches

=> pre-commit hook -> to trigger code on specific event
